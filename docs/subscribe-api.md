# Subscribe API

The subscribe function is a POST method at api.haohaotiantian.com/sub.
Example payload:
````
{
  "body": "{\"email\": \"me@testemail.com\", \"level\": \"1\"}"
}
````

Example payload for traditional characters:
````
{
  "body": "{\"email\": \"me@testemail.com\", \"level\": \"1-trd\"}"
}
````
